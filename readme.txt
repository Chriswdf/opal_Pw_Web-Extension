# Webformular Automatisierung

Dies ist ein JavaScript-basiertes Automatisierungsskript, das Daten von einer Webseite extrahiert und ein Formular automatisch ausfüllt. Es nutzt die Werte aus einem `legend`-Tag und füllt ein Eingabefeld mit diesen Werten. Anschließend klickt es auf den "Überprüfen"-Button, um das Formular zu übermitteln.
